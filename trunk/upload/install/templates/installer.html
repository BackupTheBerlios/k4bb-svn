Database Information

The database connection is an integral part of the k4 Bulletin Board. For installation purposes, k4 requires that the account information provided refers to an account with administrator permissions on the given database.

If you are wondering why there are two MySQL drivers, that is due to a newer 'improved' MySQL extension strictly for PHP 5 and up. If you are running PHP 5 and your MySQL version is greater than 4.1.1, please chose the MySQLi driver if you wish to take advantage of the additional features unique to that build.
