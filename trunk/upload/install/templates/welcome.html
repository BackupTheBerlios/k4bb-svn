General Checking

In order to safely and fully install/upgrade k4 Bulletin Board on your server, the following conditions must be met:

Databases

k4 Bulletin Board can run on several database types. The following will check if your server has at least one of the four databases installed that k4BB works with.

MySQL {$has_mysql}
MySQLi {$has_mysqli}
SQLite {$has_sqlite}
PostgreSQL {$has_pgsql}
Passed: {$db_passed}
Fileserver

In order to run k4 Bulletin Board at peak efficiency and properly, several folders on your file server must have their permissions changed. The following files must have their CHMOD permissions changed to 0777. This means that these files have read, write and remove access.

If you don't understand what CHMOD is or how to change the permissions on a file, please visit this website: Wikipedia - CHMOD

/upload/tmp (This directory and ALL sub-directories) {$has_chmod_tmp}
/upload/includes/k4bb {$has_chmod_k4bb}
/upload/templates/Descent/compiled {$has_chmod_tc}
/upload/templates/Descent/admin/compiled {$has_chmod_tac}
/upload/templates/Descent/admin/css/compiled {$has_chmod_tacc}
/upload/templates/Archive/compiled {$has_chmod_ac}
/upload/templates/RSS/ (all sub-directories of this directory) {$has_chmod_rc}
Passed: {$fs_passed}

You can choose to continue this setup whether or not you have passed the above tests. It is required that you pass the database tests; however, you might very well be adventurous and be prepared to hit a brick wall.. so who knows!
